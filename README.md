# 티카투카(Tikatuka) 최선수 어드바이저

주사위 배치 보드게임 **티카투카**에서, 현재 판 상태와 내가 굴린 눈을 입력하면
몬테카를로 시뮬레이션으로 각 수의 **승률을 추정**해 최선의 수를 추천하는 도구.

운(주사위) 요소가 있으므로 완전 탐색 대신 무작위 플레이아웃 기반으로 승률을 계산한다.

## 웹 버전 (권장)

`index.html` 을 브라우저에서 **더블클릭으로 열면** 끝. 설치/서버 불필요.

- 시작하면 **선턴(첫 차례)** 을 고르고, 선턴 플레이어가 실드 주사위로 시작한다(자기 판에만 배치).
- 나(왼쪽)·상대(오른쪽) 보드가 시각적으로 표시되고, 우세한 필드는 초록색으로 강조.
- **실시간 승률 바**: 현재 판 기준 내 승률을 상단 바로 항상 표시(주사위 굴리기 전 기준).
- **상대 차례 분석**: 상대 눈을 고르면 상대판 각 필드에 "상대가 거기 두면 내 승률"이 뜨고,
  상대 최선(=내 승률 최소) 수가 빨갛게 강조된다.
- **홀드**: 굴린 뒤 `홀드` 버튼으로 더 안 두고 상대만 진행시킬 수 있다. 버튼에 "홀드 시 내 승률"이
  표시되며 ~85% 이상이면 금색으로 강조(안전). 승리를 굳혔을 때 사용.
- **실드 주사위 구분**: 내가 깐 실드 = 파란 테두리·"나" 배지, 상대가 깐 실드 = 빨간 테두리·"상" 배지.
- **리롤 추천**: 리롤이 유리하면 초록 배너 + 리롤 버튼이 강조된다(예: `리롤 ↻ +6.6%p`).
- **내 차례**: `내 차례` 선택 → 주사위 눈(1~6) 클릭 → 각 필드에 승률 핀이 뜸
  (금색 = 추천, 빨강 = 알까기). 원하는 필드를 클릭하면 그 수가 반영됨.
- **리롤**: 눈을 고른 뒤 `리롤` 버튼 → 새 눈 선택. 이전 눈과 새 눈 **둘 중 더 좋은 쪽**을
  필드별로 표시(`[눈] 65%`). 게임당 1회.
- **알까기**: 해당 필드 클릭 시 발동 → 받은 실드 주사위 눈을 고르고, 놓을 칸을 클릭(양쪽 가능).
- **상대 차례**: `상대 차례` 선택 → 상대가 굴린 눈 클릭 → 상대가 둔 필드 클릭으로 기록.
- 하단 `판 직접 수정 / 설정`: 선턴 실드(`6s`)·시뮬레이션 횟수·새 게임 등.

| 파일 | 역할 |
|------|------|
| `index.html` | 화면/스타일 |
| `app.js` | UI 로직 (입력·렌더·클릭) |
| `engine.js` | 게임 엔진 + 몬테카를로 솔버 |

## 구현된 규칙

- 2인. 플레이어당 **3필드 × 3칸 = 9개** 주사위.
- 필드 점수: 같은 눈 n개 → `(2n-1) × 눈`. (1개=v, 2개=3v, 3개=5v)
- 승리: **3필드 중 2필드 우세** → 승. 아니면 **전체 눈 합** 비교, 같으면 무승부.
- **알까기**: 내가 놓는 필드와 *같은 위치*의 상대 필드에 같은 눈(실드 제외)이 있으면,
  그 필드를 선택 시 반드시 발동.
  - 내 주사위 + 상대의 그 눈을 모두 제거(트레이드), 실드 주사위 1개 획득.
  - 알까기 실드 주사위는 양쪽 어느 필드에나 배치 가능, 리롤 불가.
  - 다른 필드를 고르면 일반 배치(알까기 안 함).
- **실드 주사위**는 알까기로 제거되지 않음.
- **선턴 시작 실드 주사위**는 내 필드에만 배치 가능(상대 불가), 리롤 가능.
- **리롤**: 게임당 1회. 리롤해도 이전 눈을 버리지 않고 **두 눈 중 하나를 선택**.
  (선턴 실드 리롤과 공통 풀)
- **홀드**: 굴린 뒤 더 두지 않고 상대만 진행시킬 수 있음(승리 확신 시).
- **되돌리기**: `되돌리기` 버튼으로 마지막 수를 취소. 게임이 끝나면 `↻ 새 게임` 버튼 표시.

## 명시되지 않아 둔 가정 (필요 시 알려주세요)

1. **게임 종료**: 양쪽이 모두 홀드했거나 판이 가득 차면 종료 후 최종 점수 비교.
2. **알까기 발동 조건**: 상대의 같은 눈 중 *실드가 아닌* 주사위가 1개 이상일 때만 발동.
3. **상대 플레이 모델**: 시뮬레이션 속 상대는 '한 수 앞 판세'를 최대화하는 그리디로 가정.
   상대가 더 약하면 실제 승률은 표시값보다 높고, 더 강하면 낮을 수 있음.
4. 1-1-무(2필드 우세 미달) 상황도 전체 눈 합으로 판정.
